# phoneScreenshots

## Imágenes requeridas

Coloca aquí las capturas de pantalla de tu aplicación.

### Requisitos:

- **Formato:** PNG o JPG
- **Cantidad:** Mínimo 2, máximo 8
- **Dimensiones recomendadas:** 1080x1920 (vertical) o 1920x1080 (horizontal)
- **Ratio máximo:** altura/ancho debe ser como máximo 2:1
- **Tamaño de archivo:** Mantén archivos pequeños (considera conexiones lentas)

### Nombres de archivo:

```
1.png
2.png
3.png
4.png
...
```

### Qué capturar:

1. Pantalla principal / menú
2. Selección de categorías
3. Configuración de partida
4. Pantalla de juego mostrando una palabra
5. Ruleta de castigos (si es característica destacada)
6. Cualquier otra característica única

### Nota importante:

Estas capturas serán visibles en F-Droid. Asegúrate de que:
- Sean claras y legibles
- Muestren las características principales
- No contengan información personal
- Representen fielmente la aplicación
